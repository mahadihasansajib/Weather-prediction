# Weather Prediction AI Project

A comprehensive machine learning project for weather prediction, developed for university AI courses.

## Features

- **Multiple ML Models**: Linear Regression, Random Forest, Gradient Boosting, XGBoost, Neural Networks
- **Feature Engineering**: Time-based features, lag features, rolling statistics
- **Comprehensive Evaluation**: Multiple metrics and visualization
- **Interactive Demo**: Prediction and forecasting capabilities
- **Free & Open Source**: Uses only free tools and libraries
